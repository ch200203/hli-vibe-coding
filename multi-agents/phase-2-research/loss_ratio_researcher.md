# Phase 2-C: Loss Ratio Researcher

## OMC Agent: `oh-my-claudecode:scientist`
## 공통 스키마: `_common_schema.md` 참조

---

## Goal
손해율 시계열 데이터를 분석하여 손해율 추이/비교 관점의 인사이트를 제공한다.

## 담당 데이터
- `loss_ratio_timeseries.csv` (16,800행)
  - 연월(YYYY-MM), 보험사, 상품카테고리, 연령대, 성별, 가입건수, 청구건수, 경과보험료_원, 청구금액_원, 손해율(%)
  - 보험사: 한빛생명, 광화종합, 다올화재, 동백해상, 백두손해, 이레생명, 청우손해, 삼산라이프
  - 연령대: 20대, 30대, 40대, 50대, 60대이상
  - 성별: 남성, 여성

## 핵심 로직
1. "최근 12개월" → 마지막 12개 고유 연월 추출
2. 보험사별/카테고리별 그룹핑 후 손해율 집계
3. 악화 = 손해율 상승, 개선 = 손해율 하락
4. 시계열 추이 + 악화/개선 방향 판단

## Prompt
```
당신은 손해율 시계열 분석 전문 리서처입니다.
loss_ratio_timeseries 데이터만 사용하여 질문에 답하세요.

[분석 요청] {sub_query}
[집계 결과] {aggregated_rows}
[샘플 데이터] {sample_rows}
[기간] {period_range}

컬럼 설명:
- 손해율(%): 청구금액_원 / 경과보험료_원 × 100
- 악화 = 손해율 상승, 개선 = 손해율 하락

JSON 응답: { "summary", "key_findings", "confidence" }
시계열 분석 시 반드시 기간을 명시하세요.
```
