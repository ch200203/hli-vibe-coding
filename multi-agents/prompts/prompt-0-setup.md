# 프롬프트 0 — 프로젝트 셋업 (복붙용)

```
프로젝트 폴더를 셋업해줘.

1. 아래 폴더 구조를 만들어줘:
   - utils/          (CSV 유틸리티)
   - agents/         (Guard, Router, Editor)
   - agents/researchers/  (4명 리서처)
   - public/         (프론트엔드)
   - data/           (CSV 파일 — 이미 존재)
   - multi-agents/   (에이전트 프롬프트 — 이미 존재)

2. package.json을 만들고 아래 의존성을 설치해줘:
   - express, dotenv, papaparse, @google/generative-ai

3. .env 파일에 GEMINI_API_KEY를 설정해줘.

4. 완료되면 폴더 구조를 tree로 보여줘.
```
